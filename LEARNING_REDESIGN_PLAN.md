# Learning Module Redesign Plan
## Coinbase UI + Duolingo Feel

### Design Philosophy
**Goal**: Make it feel like "deja vu" - familiar Duolingo patterns but with Coinbase's professional aesthetic

### Key Changes

#### 1. Color Scheme Transformation
- **Background**: Change from bright green (#58CC02) to clean white/light gray (#F5F7FA)
- **Primary Accent**: Keep blue (#0052FF) as primary (matches rest of app)
- **Success/Completion**: Use subtle green (#10B981) instead of bright Duolingo green
- **Cards**: White backgrounds with subtle borders and shadows (Coinbase style)

#### 2. Header Redesign
- **Current**: Green background with white text
- **New**: Clean white header with blue accents
- **Keep**: Streak, XP, hearts, gems (but make them more subtle)
- **Style**: Professional badges instead of colorful bubbles

#### 3. Lesson Cards
- **Current**: Bright blue/green with heavy shadows
- **New**: 
  - White cards with subtle borders (#E5E7EB)
  - Blue accent for unlocked lessons
  - Subtle green for completed (not bright)
  - Professional shadows (0.02-0.04 opacity)
  - Keep the circular progress indicators but make them more refined

#### 4. Typography
- **Current**: Poppins (playful)
- **New**: Inter (professional, matches Coinbase)
- **Sizes**: Slightly smaller, more refined
- **Weights**: Use 600 instead of bold for headers

#### 5. Spacing & Layout
- **Padding**: More consistent (16px standard)
- **Borders**: Subtle 1px borders instead of thick colored borders
- **Shadows**: Very subtle (Coinbase style)
- **Radius**: Consistent 12-16px border radius

#### 6. Gamification Elements (Keep but Refine)
- **Streak**: Keep fire icon but in blue/orange accent
- **XP**: Keep but in subtle badge style
- **Hearts**: Keep but make them more minimal
- **Progress**: Keep circular indicators but refine styling
- **Completion**: Green checkmark but subtle

#### 7. Interactive Elements
- **Buttons**: Coinbase-style with subtle shadows
- **Hover/Tap**: Subtle elevation changes
- **Gradients**: Remove heavy gradients, use subtle ones or solid colors
